# FinStart – Корпоративный сайт

Простой, современный лендинг для компании FinStart (автоматизация бизнес-коммуникаций).

## Технологии
- HTML5, TailwindCSS, JavaScript (Vanilla)
- Адаптив, mobile-first
- Динамическая загрузка услуг и FAQ из JSON

## Структура
- /index.html – длинная главная страница (hero, услуги, процесс, отзывы, FAQ, контакты, 4-колонный футер)
- /about, /services, /contact, /privacy – внутренние страницы
- /data – JSON-файлы
- /assets – стили и скрипты

## Развёртывание
Просто скопируйте файлы на любой хостинг (Timeweb). Формы работают с локальным alert (можно заменить на PHP-обработчик).
